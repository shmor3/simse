# simse — Feature Bullets

## Core Features
- **Persistent context** — Your conversation history and project context carry over between sessions
- **Adaptive preferences** — simse learns how you work and adjusts to your style over time
- **Multi-backend support** — Connect any AI model through ACP (Agent Client Protocol)
- **Tool extensibility** — Add capabilities via MCP (Model Context Protocol) servers
- **Evolving memory** — A built-in library system that stores, organizes, and retrieves relevant knowledge

## For Directory Listings (concise)
- Context that carries over between sessions

- Connect any AI backend via ACP

## Differentiators
- Unlike ChatGPT/Claude: simse remembers your full working context across sessions
- Unlike custom GPTs: simse adapts automatically without manual configuration
- Unlike LangChain: simse is a complete assistant, not a framework to build one
- Unlike Copilot: simse works across all your workflows, not just code
